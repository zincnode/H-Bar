# H-Bar

![logo](https://raw.githubusercontent.com/zincnode/H-Bar/main/images/icon-hydrogen.png)

H-Bar means *Hydrogen-Bar*. It is a VSCode extension to show system HW/SW stats in VSCode status bar.

## Features

- CPU usage rate
- Memory usage
- Uptime
- User
- Docker
- Network speed
